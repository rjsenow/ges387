Interactive Maps by Robert John Senow

In my Interactive Cartography Class, I learned many new and wonderful techniques related to creating an interactive map. Here are some techniques that I have learned: famaliarization with HTML code, basics javascript, and web elements. I learned to create the markers & popups to the basemap, and created a Choropleth map and used data classification methods using GeoJson data. I had a chance to work with software programs called Leaflet and D3 to create a hexbin map, pie charts, donut charts, animanted path map, and Spatiotemporal (timeline slider) Proportional Symbols map. Also, I understand how to apply different techniques and functions to the map and website such as layer controls, legends, and interactive components like hover and popup with information.
